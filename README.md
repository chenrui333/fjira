# Fjira - the command line tool for Jira

![Fira](fjira.png)

## Getting started

```shell
brew install fjira
```

## Usage

```shell
Usage:
    fjira 
    fjira JIRA-TICKET
    fjira [OPTIONS]

Optional options:
    -p, --project               Search for issues withing project, example: GEN.
    -i, --issue                 Open Jira Issue, example: GEN-123.
```
